Comment installer un plugin ?

Trois étapes simples pour ajouter un plugin CPlugins à votre serveur Minecraft.

01
Téléchargez le .jar
Cliquez sur « Téléchargement direct » sur la fiche du plugin pour récupérer le fichier .jar.
02
Placez-le dans /plugins
Copiez le fichier dans le dossier /plugins de votre serveur Spigot ou Paper.
03
Redémarrez le serveur
Relancez votre serveur pour que le plugin soit chargé, puis vérifiez la config générée.
04
Besoin d'aide ?
Consultez la documentation liée à chaque plugin ou passez sur notre Discord pour du support direct.
